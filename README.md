# ✊✋✌️ Rock Paper Scissors Game (Java)

A simple console-based **Rock Paper Scissors** game built using **Java**. In this game, the player competes against the computer for **5 rounds**. The computer generates random moves, and the final winner is decided based on the total number of rounds won.

---

## 🎮 Features

- 🧑 Player vs Computer gameplay
- 🎲 Computer chooses randomly using the `Random` class
- 📝 User input using the `Scanner` class
- ✅ Input validation (only accepts 0, 1, or 2)
- 🏆 Best of 5 rounds
- 📊 Final score displayed at the end
- 🤝 Draw detection for each round

---

## 🛠️ Technologies Used

- Java
- IntelliJ IDEA
- Object-Oriented Programming (OOP)

---

## 🎯 Game Rules

| Number | Choice |
|--------:|--------|
| 0 | 🪨 Rock |
| 1 | 📄 Paper |
| 2 | ✂️ Scissors |

### Winning Rules

- 🪨 Rock beats ✂️ Scissors
- 📄 Paper beats 🪨 Rock
- ✂️ Scissors beats 📄 Paper

---

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/Vikashkush2026/rock-paper-scissors-java.git
```

2. Open the project in IntelliJ IDEA.

3. Run the `Game_Rock_Paper_Scissor.java` file.

4. Enter your choice:

```
0 = Rock
1 = Paper
2 = Scissors
```

5. Play 5 rounds and see who wins!
